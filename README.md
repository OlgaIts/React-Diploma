# Дипломный проект курса «React»

Дипломный проект представляет собой интернет-магазин обуви.
Техническое задание находится [здась](https://github.com/netology-code/ra16-diploma) 


```
Запуск:
npm install
npm run dev

_____________________

Запуск сервера:
cd backend
npm install
npm run watch
```
